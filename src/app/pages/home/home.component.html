MEAN Checklist

This is an example checklist application written with MongoDB, Express, Angular CLI, Node.js, Gulp.js, and Bootstrap 4. It has an authentication API tested with POSTMAN and boasts a full CRUD for checklists with tasks.

Create a Checklist  

  Node.js + Angular CLI

Angular CLI and Node.js are the best of friends (most of the time). This app demonstrates the minimal modifications needed to make a Node.js API properly communicate with Angular 2+'s structure.

  Server Authentication + JWT

This application runs on a MongoDB database. It authenticates through Passport.js and use JWT tokens to approve requests. Node.js has evolved over the years to provide a simple yet secure boiler plate for APIs.

  Live Mock API

The best API implementations provide a pre-populated database for testing. Through a simple command of npm run serve-db-mocks this project will generate a database for API testing. Perfect for offline development.
